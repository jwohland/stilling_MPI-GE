Land use data used in CMIP5 experiments, downloaded in January 2021 from 

https://luh.umd.edu/data.shtml#LUH1_Data

(historic data, LUHa.v1). To save diskspace, only update states of gothr and gsecd are kept (i.e. the folder lu is deleted).
All years prior to 1850 are also deleted.

IMAGE, MiniCAM and MESSAGE data downloaded on Feb. 1st from the same web portal.
